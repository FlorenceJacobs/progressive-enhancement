D'où vient Markdown ?

Origine

Markdown a été théorisé par John Gruber en 2004.

  • Son but initial était de concevoir un outil de conversion texte-vers-HTML pour ceux qui écrivent pour le web.
  • Avec un langage qui soit facile à lire, facile à écrire, basé sur un format texte, et facile à convertir en HTML.

et facile à convertir en HTML

Vive le Markdown !
